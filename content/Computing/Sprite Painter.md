
The **Sprite Painter** is a tool used to create textures for the Commodiesel. You can access it by using [this link](https://jimensions.vercel.app/Sprite%20Painter.html), or by downloading the HTML file directly from GitHub, which you can find [here](https://github.com/Gassandrid/crdg.xyz/blob/v4/content/Computing/spritepainter2.html).
