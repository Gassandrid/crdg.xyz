**Compilers** are a special class of program for writing code for the [[Commodiesel 75]]. They are completely external from the game itself, and are used to take advantage of out of game code and media to create programs for use in the game. They typically take some source data (say an image, midi file, or text file) and _compile_ them into Commodiesel 75 machine code that can be run in the game.

This allows you to do things like import images, music, and other media into the game, as well as write programs in higher level programming languages that are then compiled down to Commodiesel 75 machine code.
