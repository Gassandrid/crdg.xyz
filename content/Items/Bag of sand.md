The **Bag of sand** is a Tool that can be found at the [[Gas Station]].

---
# Usage
When holding this item and clicking, it will create 5 small sand parts, these sand parts can be heated in the furnace in the hidden room of [[Places/Cathedral|Cathedral]] to create glass, aside from this they have no special properties.
If one has the Cardboard item, this can be used to create infinite debris to make cardboard with.
