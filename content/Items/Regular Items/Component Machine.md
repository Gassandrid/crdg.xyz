---
## Usage

**Component Machine** allows its owner to create numerous component. (can be found in [[content/Components/Regular Components/index|Regular Components]]). Player is able to either spawn component using roulette which rolls for 5 seconds and spawns a random component,  or type the name of desired component and spawn it for double the cost.

> [!Tip]
> - Unequipping **Component Machine** mid-spawning will result in component spawn position be at the spot where it was unequipped (can be refreshed by re-equipping), which is useful if you want to go somewhere while spawning component!
> 
> 
> - Spawning component using its name costs 2 points, while spawning random one is only 1.
