The **Bag of sand** is a Tool that can be found at the [[Gas Station]]. It does not have an icon. It is also one of the only items in the game to be able to get multiple of.

---
## Usage

When holding this item and clicking, it will create 5 small sand parts. These sand parts can be heated in the furnace in the hidden room of [[Places/Cathedral|Cathedral]] to create glass, but aside from this they have no special properties.

If one has the cardboard item, this can be used to create infinite debris to make cardboard with.
