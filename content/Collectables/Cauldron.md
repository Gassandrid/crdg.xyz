The **Cauldron** is a miscellaneous-type blimp decoration added on August 17th, 2025 during the Big Blimp update. It acts similar to the [[Fab]], using a combination-based system to create a product. 

## Usage

To use the **Cauldron**, simply insert a valid ingredient. A list of all valid ingredients can be found below. Note that all liquids work as an ingredient, and that not all components work as ingredients.
