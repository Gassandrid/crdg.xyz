title: Regular Components
---
**Regular Components** can be obtained by spawning them using [[content/Items/Regular Items/Component Machine|Component Machine]], with every component serving different purpose.

Components are the primary in game objects that can be used for building with [[content/Items/Regular Items/Glue|Glue]].
