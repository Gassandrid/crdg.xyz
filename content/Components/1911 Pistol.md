The **1911 Pistol** is a [[crdg.xyz/content/Features/Manifests|Manifest]] **Component** obtainable by public server **Seed Trees** and **Fishing**.  
Overall, it is a better version of the [[crdg.xyz/content/Components/Gun|Gun]] component with no reload cooldown. 

## Usage

- Combining it with a [[crdg.xyz/content/Components/seeker|seeker]] allows you to make a simple turret. However, it’s able to attack you.
- If you combined it with a [[Trigger]], you’d be able to hold the gun, allowing you to shoot at anything.
