The **Worldspike** is a miscellaneous-type [[content/Components/Alien Technology/index|Alien Technology]] added on November 28th, 2024. It is an upgraded version of the [[Spikes]] component, being able to anchor while airborne and simultaneously being a gyroscope.

## Usage

To use the **Worldspike**, activate it. The middle glass part will anchor itself, but the triangular pentagon shape bordering it will start spinning, indicating that the **Worldspike** is stuck. However, activating it also turns it into a hinge, meaning that it is able to be rotated in the sideways axis.

> [!Tip] Full Anchoring
> By using two **Worldspikes**, you are able to fully anchor builds due to the second **worldspike** anchoring the hinge from the first one.
