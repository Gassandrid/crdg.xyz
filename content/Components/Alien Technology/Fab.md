The **Fab**, also known as the **Fabricator** or the **Printer** is a **Miscellaneous**-type [[content/Components/Alien Technology/index|Alien Technology]] added on **November 28th, 2024** during **Black Friday 2024**. It is similar to the [[Cauldron]], using an input system (in this case, a code-based system) and fuel to create a **slab**, also known as a **tile**.

## Usage

To use the **Fab**, create an [[Uranium Rod]] from the [[content/Items/Obtainable Items/Component Machine]] and insert it through the hole on the back. The [[Uranium Rod]] should now be dug up into the hole and should've played a sound indicating so. 

> [!Tip]+ Uranium Rod Stacking
> You're able to glue uranium rods together in a line order to (in a way) over-fuel the **Fab**. This is especially useful for fabricating slabs like the **Soil Slab** in bulk.
### Input System


The **Fab** uses a **code-based system** comprised of pipes (|), underscores (\_\) and asterisks (\*\) to create any slab of your choice. Note that the following tutorial assumes you are facing the hole side of the **fab**.

- To input a **pipe** (|), simply click or tap the right side of the printer. The side where the [[Uranium Rod]] is placed should show said **pipe** shown on top of the side.
- To input an **underscore** (\_\), click the right side of the printer again. You need a delay of at least **one second** (depending on the tick speed of the server) after first inputting a **pipe**.
	- You are unable to input an **underscore** as the first input of a code. This means that for example codes like `_|_|||` are impossible to create.
- To input an **asterisk** (\*\), tap the left side of the **fab** exactly **five times**. This input is only restricted to the first input of a code, meaning codes like `I_*III` or `I*III` are impossible.
	- This input is able to be followed by an **underscore**. However, you have to wait a delay of at least **three seconds** to do so. This means codes like `*_I_III` are possible.
- To clear your code, either input more than 10 characters of the code or press the **left side** of the **fab**.

> [!Tip]+ Cart Control Pressers
> [[cart control presser|cart control pressers]] are able to be used as an input for the **Fab**. This means you're able to place five [[cart control presser|cart control pressers]] on the **clear side** of the **Fab** to simulate an asterisk (\*\) press with only one button.
### Slabs

The printer has a bunch of combinations that give flat blocks called **slabs**. All slabs have a different code assigned to them, and they can vary in colour, material, texture, size[^1] and usage. There are currently **forty** slabs (including six that have an asterisk in their code) in the game, with the last one added on **January 8th, 2025**. Most of these slabs are for building purposes, but some have different use cases. Below is a list of all forty of them with their respective codes.
