Component machine allows player to summon various [[Components]] using points, which takes 15 seconds to regenerate. Player can use points to either roll the component list to get random component from the [[Components]] list (consumes 1 point), or by typing in the component name manually (cosnumes 2 points). Best used with [[glue]].

More information can be found on [[content/Items/Obtainable Items/Component Machine|Component Machine]] page!
