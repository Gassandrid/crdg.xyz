Because many of you may not have worked with **Git** or **Github** before, we think it best there be a guide on how to add *your changes* to the main website.

---

## Steps

This will assume you already have set up a **Github** account, and that you have the CRDG.xyz wiki open.

### Fork the Repository

Click the **fork** button on the main page, this will prompt you to make your own version of the repository that you can edit.

![[Screenshot 2025-07-26 at 11.57.16 AM.png]]

### Make You Changes

Now that you have you own version of the wiki, you can do two things:

#### Make Your Changes Directly in GitHub

Click *content* to edit the wiki source

![[Screenshot 2025-07-26 at 11.59.25 AM.png]]

*Click the edit button*

![[Screenshot 2025-07-26 at 12.00.16 PM.png]]

**OR**

#### Editing With Obsidian

Click the *code* button

![[Screenshot 2025-07-26 at 12.01.32 PM.png]]

And then copy the **URL** of your repo

![[Screenshot 2025-08-15 at 1.42.37 PM.png]]

Then, **IF YOU ARE ON WINDOWS**, you can clone the files onto your own computer (**make sure to choose a good spot for it**):

![[Pasted image 20250815134329.png]]

Then, go to [[Using Obsidian to Make Changes]] to learn how to use it to add changes.

### Adding Your Changes to the Main Wiki

Once you have added and **committed** your changes ( please look this up its very easy ), you can now make a **pull request**:

First go to the **pull requests** tab on YOUR fork:

![[Screenshot 2025-07-26 at 12.04.33 PM.png]]

Then click **new pull request**:

![[Screenshot 2025-07-26 at 12.04.49 PM.png]]

You should see **Your fork** compared to the main repository, and you can configure/write about what you updated:

![[Screenshot 2025-07-26 at 12.05.15 PM.png]]

Then make it, and you are done! It will be reviewed by a maintainer and then added.
