Super glue is an admin item that is given only to admins of CRDG, which
behaves as a buffed [Glue](content/Old%20Wiki/Glue.md "glue"){.wikilink}. It is a glue bottle
with a red tint, resembling gorilla glue.

## Function

Superglue behaves similarly to standard glue, with a few key
differences. For one, it is able to glue things that aren\'t meant to be
glued, such as players, while being impossible to remove after it is
glued. It is unknown whether the superglue tool can unglue superglue,
but the standard glue tool is unable to.
