Assemblies are common structures composed of more than 1 part which
serve a purpose which no single part is able to fulfill.

Articles written for assemblies should contain information about their
use cases as well as a detailed tutorial for building it. If an assembly
has a number of different components you can use to build it, then you
should detail which parts are able to be swapped. If it\'s constructed
in a completely different fashion but serves the same utility, then a
separate article should still be written for it.
