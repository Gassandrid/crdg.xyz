The [component
machine](Component_Machine "component machine"){.wikilink} is a item
that can be achieved by buying the component machine gamepass for 220
Robux. When used a menu opens up that can be used to get components by
either typing in the name or pressing \"get an thing\".

## Description

this handheld machine creates random components from thin air, from
springs to rockets! best used with glue
